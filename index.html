Acerca de mi

Soy un desarrollador de videojuegos especializado en Unity3D, con una fuerte inclinación hacia el diseño e implementación de interfaces gráficas (UI/UX). Me apasiona crear experiencias intuitivas, limpias y funcionales que conecten al jugador con el mundo del juego de forma natural.

Disfruto trabajar en la intersección entre lo visual y lo técnico, donde cada pantalla, botón y flujo tiene un propósito.

Creo firmemente que una buena interfaz no solo organiza información: cuenta una historia y potencia la jugabilidad.

Siempre busco mejorar mis procesos y adaptarme a las necesidades de cada proyecto, sin perder de vista la experiencia del usuario como eje central del desarrollo.

Portfolio


    Realm of the Storm — UI Programmer

    "Realm of the Storm" es un ambicioso RPG naval multijugador en tiempo real desarrollado por un equipo de 4 personas. El proyecto combina exploración marítima, batallas estratégicas (1vs1 y 3vs3), sistemas de progreso tipo RPG y economías complejas dentro de un mundo persistente.

    Mi rol principal fue el desarrollo de la interfaz de usuario, utilizando UI Toolkit de Unity, una tecnología basada en estándares web moderna que permite crear interfaces dinámicas y escalables. Trabajé estrechamente con un diseñador digital, integrando sus diseños en Unity y haciendo funcional cada elemento mediante programación, enlazando los datos del backend provistos por PlayFab.

    Entre las tareas clave que realicé se destacan:

    • Implementación de interfaces complejas como:
      • - Sistema de inventario de media-alta complejidad con objetos crafteables y equipamiento con estadísticas personalizadas.
      • - Sistema de crafteo para crear consumibles, armas, armaduras y mejoras para barcos.
      • - Gestión de barcos, con soporte para slots de equipamiento y consumibles.
      • - Interfaz de batallas navales en tiempo real y multijugador.
      • - Paneles de divisas y sistema de cambio de monedas (oro, gemas), integrado con la pasarela de pagos Stripe.
      • - Sistema de chat en tiempo real, lista de amigos, grupos de exploración y guilds.

    Me enfoqué en la integración fluida entre UI y backend, asegurando la correcta visualización e interacción del usuario con los datos del servidor y mejorando la experiencia del jugador.




    Neraverse Legendary — UI Programmer

    "Neraverse Legendary" es un videojuego multijugador de carreras en tiempo real desarrollado con Photon Fusion, en el que los jugadores compiten en frenéticas carreras, personalizan vehículos y avatares, y forman parte de una comunidad competitiva mediante clanes y chat en línea.

    En este proyecto, trabajé como programador de interfaces de usuario, utilizando el sistema clásico de UI de Unity. Mi responsabilidad principal fue transformar los diseños visuales en interfaces funcionales, integrándolos con el backend proporcionado por PlayFab.

    Entre mis tareas destacadas se incluyen:

    • Implementación e integración de paneles de usuario como:
      • - Sistema de personalización del vehículo: ruedas, chasis, antenas, alerones, entre otros accesorios visuales.
      • - Sistema de personalización del conductor: con skins aplicables a distintas partes del cuerpo como cabeza, manos, casco, armas, etc.
      • - Tienda de vehículos: compra de nuevos coches y accesorios usando divisas obtenidas dentro del juego.
      • - Gestión de clanes y sistema de chat en tiempo real para la comunicación social entre jugadores.

    Mi trabajo se centró en conectar eficientemente cada interfaz con la lógica del juego y los datos almacenados en PlayFab, asegurando una experiencia de usuario fluida y funcional tanto en personalización como en sistemas sociales y económicos del juego.
